# Fast minimal navigation system for android

Motorcycle navigation for low budget devices with Android 14+
Works fully offline, no online service needed. Available offline data: Map, POI and routing data.

This app is simple, it can:
- Search for Address and POIs
- Save your favorite Locations and Rides
- Import / Export GPX routes and tracks
- Navigate on routes and tracks
- Find fuel stations, restaurants or hotels on the route
- Stay informed about speedlimits, time and distance to stops and route or track end

This app delivers all the essentials. Not fancy, but fast.
